# TypeScript Development
* write and run test in TypeScript
* building browserify

# Env
* Node.js v5.1
* TypeScript v1.7.5

# Usage

## first
```
$ npm i
```

## test
```
$ npm test
```

## watch
```
$ npm run watch
```

## bundle
```
$ npm run bundle
```

## At Browser
```
<html>
  <body>
    <script src="./dist/app.js"></script>
  </body>
</html>
```
